# 📞 Smart Call Tracker

An Android application that tracks call history, generates monthly reports, and exports reports as PDF.

---

## ✨ Features

- 📞 Call History Tracking
- 📅 Monthly Reports
- 📊 Statistics Dashboard
- 📄 PDF Report Export
- 🔒 PIN Security
- 🔐 Biometric Authentication
- 🤖 SmartGPT Assistant
- 🎵 Background Music
- 📆 Calendar View
- 💾 Room Database
- 🔔 Notifications

---

## 🛠️ Tech Stack

- Kotlin
- Android Studio
- Room Database
- WorkManager
- Material Design
- RecyclerView

---

## 📂 Project Structure

```
app/
database/
adapter/
model/
security/
worker/
```

---

## 🚀 Installation

1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run the application
